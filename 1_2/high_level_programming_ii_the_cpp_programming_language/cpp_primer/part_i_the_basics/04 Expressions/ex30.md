Using Table 4.12 (p. 166), parenthesize the following expressions to match the default evaluation:

(a) `sizeof x + y`<br>
`(sizeof x) + y`

(b) `sizeof p->mem[i]`<br>
`sizeof (p->mem[i])`

(c) `sizeof a < b`<br>
`(sizeof a) < b`

(d) `sizeof f()`<br>
`sizeof (f())`<br>
if `f()` returns nothing, then the result is undefined.
